# WISP Pago

## Project Overview

WISP Pago is a comprehensive billing and reporting system designed for Wireless Internet Service Providers (WISPs). The system streamlines the billing process, provides insightful reporting, and enhances customer management for effective service delivery.

## Features
- ✅ Automated billing cycles
- ✅ Detailed usage and financial reports
- ✅ User-friendly interface
- ✅ Integration with payment gateways
- ✅ Customer management tools
- ✅ Dashboard with analytics

## Tech Stack
- **Frontend**: React
- **Backend**: Node.js + Express
- **Database**: PostgreSQL
- **ORM**: Sequelize

## Project Structure
```
wisp_pago/
├── backend/          # Node.js API
├── frontend/         # React application
├── database/         # Database schema and migrations

```
